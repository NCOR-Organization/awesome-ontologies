FAQ: Submit a new ontology

How do I submit my ontology?

To submit a new ontology for inclusion in the OBO Foundry, you will need to:

  1. Review the OBO Foundry requirements and technical details document for membership expectations and submission instructions.
  2. Submit your request to our GitHub tracker, using the New Ontology Request template. Note: please see the New Ontology Registration Instructions for guidance.
  3. Send an email to obo-discuss with your request to allow community feedback (you may need to apply to join the obo-discuss group first.) Be sure to include the link to the issue you created in step 2.
  4. (Optional): If you would like to join the obo-community Slack, please indicate this in your email.
